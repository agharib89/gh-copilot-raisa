About This Project

A demonstration of GitHub Copilot capabilities through a Python Flask application.

Project Overview

This project showcases how GitHub Copilot can accelerate development and improve code quality through AI-powered suggestions. It's built using Python Flask and demonstrates various Copilot features including code completion, test generation, and documentation.

Key Features

  • Comprehensive Copilot Configuration: Pre-configured instructions, prompts, and agents to maximize Copilot's effectiveness.
  • Best Practices: Follows Python PEP 8 and PEP 257 conventions with type hints and comprehensive docstrings.
  • Security-First: Built-in security guidelines and best practices for secure application development.
  • Testing Standards: Pytest-based testing with high coverage standards and comprehensive test examples.
  • CI/CD Ready: Includes GitHub Actions workflow for automated testing and linting.

Getting Started

Installation

git clone https://github.com/agharib89/gh-copilot-raisa.git
cd gh-copilot-raisa
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
pip install -r requirements-dev.txt

Running the Application

python src/app.py

Then visit http://127.0.0.1:5000 in your browser.

Running Tests

pytest
